#  INSTALL / UPDATE DOWNFLOWGO PROC

- The configuration file CONF/PROCS/DOWNFLOWGO.conf must be updated to match any changes in config_downflow.ini
- Check and modify parameters such as "PATHS_DEM" (model selection)
- Do the same with CODE/tplates/PROC.DOWNFLOWGO
- Click on "PROCS" in the WebObs menu and create a new PROC with the template "PROC: DOWFLOWGO". Don't forget to select a "Domain" before saving.
