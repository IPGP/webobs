- sefran, MC3
- all proc outputs: gridmaps, locastat, genplot, ...
- proc requests and results display
- conf editors (grid, node)
- navigation into menu, grids, nodes, feature links
- text content edit+display (grids, nodes, wiki page)
- events edit+display (grids and nodes)
- upload/delete image and automatic thumbnails (grid event, node photo/scheme/doc, node event and sub-event, a genform image input)

== make the package

- git pull (master branch)
- update mods.html (if new scripts)
- update .tex user manual (version + date)
- update release-notes.md (under development section ==> x.y.z)
- commit all modified files + git push (!)
- compile:
    - user manual (LaTeX)
    - i18n (make)
    - Matlab (make)
- PACK/build-mac x.y.z (for MacOS: needs packages md5sha1sum and gnutar)
- test SETUP/setup on a dev VM
- tag new release + notes + upload package .tgz + PDF user manual
- update index.md + commit
- send an email to webobs-users@services.cnrs.fr
